# data structure in python

## list --  super important
    - mutable
    - indexed
    - ordered
    - duplicates
## tuple
    - immutable
    - indexed
    - ordered
    - duplicates
## set
    - mutable
    - unindexed
    - unordered
    - unique values
## dict --super important
    - mutable
    - unindexed
    - ordered
    - only unique keys
    - can contains duplicates values
